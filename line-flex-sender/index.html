⚠️ 事前準備：CORS制約の回避（検証用ブラウザの起動）

このツールはブラウザから直接 g123 の Broadcast API にリクエストを送ります。通常のChromeではCORSエラーが発生するため、以下の手順で検証用Chromeを起動してから使用してください。

  1. キーボードの Win + R を押して「ファイル名を指定して実行」を開く。
  2. 以下のコマンドをコピーして貼り付け、「OK」を押す。
    chrome.exe --user-data-dir="C:\ChromeDev" --disable-web-security --disable-site-isolation-trials
  3. 新しく開いたChromeでこのツールのURLを開きなおす。

※このChromeでは通常のネットサーフィンをしないでください。

⚠️ 事前準備：CORS制約の回避（検証用ブラウザの起動）

ブラウザから直接LINEのAPIを叩くため、通常のChromeでは通信がブロック(CORSエラー)されます。以下の手順でセキュリティ設定を解除した検証用Chromeを起動してからこのツールを使用してください。

  1. キーボードの Win + R を押して「ファイル名を指定して実行」を開く。
  2. 以下のコマンドをコピーして貼り付け、「OK」を押す。
    chrome.exe --user-data-dir="C:\ChromeDev" --disable-web-security --disable-site-isolation-trials
  3. 新しく開いたChromeでこのツールのURLを開きなおす。

※LINE以外への通常のネットサーフィンにはこのChromeを使用しないでください。

※「送信」ボタンを押すと、登録した全アカウントから順番に送信されます。
※ユーザーIDはLINE Developersコンソールの「チャネル基本設定」一番下で確認できます。
※上限を指定した場合、LINEの仕様によりNarrowcast APIを使用します。
